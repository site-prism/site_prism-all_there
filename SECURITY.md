# Security Policy

## Supported Versions

`site_prism-all_there` is only supported in the latest major version of the gem (Security updates or otherwise)

| Version | Supported          |
|---------|--------------------|
| 3.x     | :white_check_mark: |
| < 3.x   | :x:                |
