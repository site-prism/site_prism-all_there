# Security Policy

## Supported Versions

`site_prism-all_there` is fully supported in the current major version of the gem (Security updates and otherwise)

| Version | Supported          |
|---------|--------------------|
| 3.0.x   | :white_check_mark: |
| < 3.0.x | :x:                |
